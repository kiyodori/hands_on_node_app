# 不要なファイルを削除
実装を進める前に、デフォルトで作成されている`routes/users.js`に関連するファイル・記述を削除します。

```
$ rm routes/users.js
```

`app.js`からも関連する記述を削除します。

```app.js
// これらの行を削除する
- var users = require('./routes/users');

- app.use('/users', users);
```
