# 2. APIにアクセスしてデータを保存しよう
今回はAPIにアクセスしてデータベースにデータを保存するところまでやります。

## データベースを作成しよう
sequelizeとMySQLを使います。
まずはインストールしましょう。

※ MySQLはMacにインストールされている前提です。

```
# デフォルトでインストールされているMySQLを削除します
$ yarn remove mysql

$ yarn add sequelize
$ yarn add mysql2
```
